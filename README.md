# 🤖 Prolog Programming for Artificial Intelligence & Machine Learning

Welcome to this repository! 🚀

This repository focuses on learning and implementing **Prolog**, a logic programming language widely used in Artificial Intelligence for knowledge representation, reasoning, and expert systems.

## 📚 About Prolog

Prolog (Programming in Logic) is a declarative programming language based on formal logic. Instead of specifying how to solve a problem, programmers define facts, rules, and relationships, allowing the system to infer solutions automatically.
